Приближённый двоичный поиск
В этой задаче нужно применить функцию std::lower_bound и итераторы
для быстрого поиска ближайшего элемента в отсортированном массиве.

Формат ввода
В первой строке входных данных содержатся натуральные числа n и k,
не превосходящие 100000. Во второй строке задаются целых n чисел
первого массива, отсортированного по неубыванию, а в третьей строке –
k целых чисел второго массива. Каждое число в обоих массивах по
модулю не превосходит 
2 * 10^9. Второй массив, в отличие от первого, не отсортирован.

Формат вывода
Для каждого из k чисел выведите в отдельной строке число из
первого массива, наиболее близкое к данному. Если таких несколько,
выведите меньшее из них.
